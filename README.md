# Shemabuds - Handcrafted Gifts & Decorations

A beautiful e-commerce website for Shemabuds, specializing in handmade bouquets, gifts, and decorations.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 🛠️ Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Firebase (Firestore + Auth)
- Framer Motion
- React Router DOM

## 📱 Features

### Customer Features
- Mobile-first responsive design
- Tab-based mobile navigation (Home, Collections, Offers, Cart, Settings)
- Product browsing by categories
- Product customization
- Shopping cart
- Dual ordering: Direct order or WhatsApp
- Order success modal with confetti animation

### Admin Features
- Secure admin dashboard at `/admin`
- Product management (add/edit/delete)
- Order management
- Featured products toggle
- Real-time Firebase sync

## 🔥 Firebase Setup

1. Create Firebase project at https://console.firebase.google.com/
2. Enable Firestore and Authentication
3. Update `src/firebase.ts` with your config
4. Set Firestore rules to allow read/write
